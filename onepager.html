Claude Design とは？／全体像

ひとことで言うと 「言葉で頼むとデザインを作ってくれる Claude」。LP・スライド・1枚資料・プロトタイプ・UI を、会話で作って整えられます。あなたは“監督”、Claude が“制作スタッフ”。専門ソフトの操作は不要です。✅

中身は「デザインシステム（ルール集）」と「プロジェクト（成果物）」の2層で考えるのがコツ 🔶

Code or 各ツール
こんな悩み → こう頼むだけ
  • 「資料を見栄えよく」→「落ち着いた印象のスライドにして」
  • 「毎回トンマナがバラつく」→「うちのリポジトリを読み込んでルール化して」
  • 「そのまま実装に渡したい」→「ハンドオフにまとめて」→ Codeで受取
すぐ試せるプロンプト
# コピペで試せる例 > {自分のサービス}の紹介LPを1枚作って。ターゲットは{誰}、 トンマナは「信頼感・余白広め」。構成は ファーストビュー → 課題 → 特徴3つ → 事例 → 申し込み の順で。
実践ステップ
  • 1まず小さな1枚を会話で作る
  • 2色・文字・余白を整える／canvasでドラッグ調整
  • 3気に入った“ルール”をデザインシステムに保存
  • 4再利用し、必要なら Claude Code に渡して実装
